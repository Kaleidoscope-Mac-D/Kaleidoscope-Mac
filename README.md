## What Is Kaleidoscope for Mac

Kaleidoscope mac is a file and image comparison app that spots differences in text and image files
and presents them in a clear, single-window workflow. Rather than one generic diff view, it splits
into two purpose-built engines — Text Scope and Image Scope — plus a set of workflow tools and
version-control hooks that tie everything together. This guide takes each feature in turn so you can
see exactly what kaleidoscope for mac does and how deep it goes.

---

## Single-Window Workflow

The foundation of kaleidoscope mac is its single-window design. Every comparison is a tab, and that
one decision shapes the whole experience.

- **One tab per comparison**: Each job is isolated in its own tab, so you can keep several
  comparisons open without confusing them.
- **File Shelf**: A dedicated shelf lets you jump quickly from file to file within a comparison,
  which is invaluable when a tab holds several versions.
- **Path Bar**: The Path Bar always shows where each file lives on disk, so you never lose track of
  which copy you are actually looking at.
- **Keyboard shortcuts**: Every action has a shortcut, making a file compare mac session something
  you can drive almost entirely from the keyboard.

Because comparisons also stay always up to date — auto-refreshing whenever a file changes — the tab
in front of you is never stale.

## Text Scope

documents.

- **Three layouts**: Choose Blocks for a side-by-side block view, Fluid for a flowing comparison, or

For code review, contract redlines or documentation updates, Text Scope turns a raw diff into

## Image Scope

Image Scope gives kaleidoscope mac download a capability most diff tools lack entirely: pixel-level
image comparison.

- **All popular formats**: Compare JPEG, TIFF, PNG, PSD and more without converting anything first.
- **Four connected layouts**: Switch between Two-Up, One-Up, Split and Difference views, each linked
  so panning and zooming stay in sync.
- **Comprehensive color support**: RGB, CMYK and LAB color, alpha channels and images of any size are
  all handled.
- **Multi-touch**: Smooth scrolling and pinch-to-zoom let you inspect fine detail naturally on a
  trackpad or Magic Mouse.
- **Two difference-mask modes**: Spot either absolute or relative changes between two images,
  depending on what you are checking for.
- **Customizable environment**: Adjust the background color, the difference mask and the checkerboard
  settings so transparency and subtle changes are easy to see.

Designers and QA reviewers use Image Scope to catch a one-pixel shift, a color drift or an alpha edge

## Frequently Asked Questions

**What is the difference between Text Scope and Image Scope?**
Text Scope compares text files — plain text, source code, HTML and imported .doc and .rtf documents —
while Image Scope compares images such as JPEG, TIFF, PNG and PSD. Kaleidoscope for mac chooses the
right one for the files you open.

**Which text layouts should I use?**
Kaleidoscope mac offers Blocks, Fluid and Unified. Blocks and Fluid show side-by-side comparisons,
while Unified gives a compact single-column diff — pick whichever reads best for the file at hand.

**How does Image Scope show differences?**
It offers Two-Up, One-Up, Split and Difference layouts plus two difference-mask modes, so
kaleidoscope mac download can reveal either absolute or relative changes between two images.

**Does it handle transparency and color spaces?**
Yes. Image Scope supports RGB, CMYK and LAB color plus alpha channels, and you can tune the
checkerboard and mask settings to make transparent areas easy to read.

**Can it act as my Git diff tool?**
It can. Download kaleidoscope for mac integrates with Git, Mercurial, SVN and Bazaar as the diff and
merge viewer, so version-control changes open in the same clear comparison view.

**Will comparisons update automatically?**
Yes. Kaleidoscope for mac auto-refreshes each comparison whenever a file changes on disk, keeping the
file compare mac view current without any manual reload.
